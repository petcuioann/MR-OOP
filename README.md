# RO Programare Orientată Obiect - Matematică (Linia Română)

Acest repository conține laboratoarele, proiectele și exercițiile rezolvate pentru cursul de Programare Orientată Obiect (OOP) din cadrul Facultății de Matematică și Informatică, UBB. Materialele acoperă concepte fundamentale precum încapsularea, moștenirea, polimorfismul, utilizarea șabloanelor (templates), gestiunea memoriei și mecanismele standard din C++ (STL). Scopul acestui proiect este de a oferi studenților un ghid practic pentru înțelegerea și aplicarea eficientă a paradigmei orientate obiect în rezolvarea problemelor matematice și computaționale.
